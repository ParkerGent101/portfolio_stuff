Packet Sniffer

The code utilizes the Scapy library to sniff packets on a specified network interface. The sniffer can capture TCP, UDP, and ICMP packets and provides options for the user to choose the type of packets to capture. The script includes logging functionality to record packet details in a log file for later analysis. It also maintains statistics such as the total number of packets captured and the most frequent source IP addresses. Additionally, the script offers a visualization feature using Matplotlib to display a bar chart of the packet count for each unique IP address.
